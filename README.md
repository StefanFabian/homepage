# Building and running the website locally

1. [Install jekyll](https://jekyllrb.com/docs/installation/)
2. (Optional) Update gems including bundle
   ```
   sudo gem update
   ```
3. Install dependencies
   ```
   bundle install
   ```
4. Run
   ```
   bundle exec jekyll serve
   ```
